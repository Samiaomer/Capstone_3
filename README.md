# Capstone 3: Sales Analysis with Power BI

## Overview
This project analyzes sales data for a fictional bookstore chain selling books across various authors and categories. The goal is to create a Power BI report that highlights key insights, trends, and recent sales performance for a specific region, to be presented at a management team meeting.

## Analysis Summary
The Power BI report includes:

1. Data Loading – Connecting to source data and importing tables into Power BI
2. Data Modeling – Creating relationships between tables and defining custom measures
3. Data Transformation – Cleaning book titles and handling mismatches between tables using Power Query
4. Visualizations – Creating a line chart, column/bar chart, pie/donut chart, and a list of top-selling books
5. Custom Measure – Defining a "Book Sales" measure using DAX functions like CALCULATE, USERELATIONSHIP, and TREATAS
6. Report Layout – Designing at least two report pages with descriptive headers

## Tools & Technologies

* Microsoft Power BI Desktop
* Power Query Editor
* DAX Data Analysis Expressions

## Key Deliverables

1. Power BI .pbix file containing:
   - Cleaned and transformed data tables
   - Required report visualizations:
     - Line chart showing sales trends over time
     - Column or bar chart showing sales performance by category
     - Pie or donut chart showing relative sales for each state in the region
     - List of top-selling books, including author names, for the region (general audience, excluding textbooks)

2. Documentation of data cleaning and transformation steps performed in Power Query

3. Insights and recommendations based on the sales analysis, tailored for the regional sales team and management
